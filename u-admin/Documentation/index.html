Ultra Admin UI Template

HTML Structure

This template is fully responsive, modern, admin UI template. The general structure of all the pages are basically the same and can be accessible through the blank.html page.
To create a new page in this theme copy the blank.html page 


                                                           <div class="content-body">
                                                                <div class="row">
                                                                    /*Content*/
                                                                </div>
                                                            </div>

Start coding from the point that says /* Content*/
